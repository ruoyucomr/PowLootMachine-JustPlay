Prerequisites (Windows)
1. Rust toolchain (stable) with cargo
2. Visual Studio 2022 Build Tools with C++ workload
3. CMake (in PATH)
4. NVIDIA CUDA Toolkit (for GPU build)
5. PowLoot WASM asset (powloot_wasm.wasm-0002330e)

Optional environment
- CUDA_PATH should point to your CUDA install, e.g.
  C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v13.1
- POWLOOT_WASM can override the WASM file path, e.g.
  D:\PowLoot\static\wasm\powloot_wasm.wasm-0002330e

Notes
- If you only build CPU mode, CUDA Toolkit is not required.
